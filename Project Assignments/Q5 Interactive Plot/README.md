Requirements: Python Installed
Step 1: Install the zip file from the github repository
Step 2: Extract the file somewhere of your choice
Step 3: Go to the /Project Assignments/Q5 Interactive Plot/ folder
Step 3: Open a terminal in the folder by right clicking in the folder and clicking "Open in Terminal"
Step 4: Run "py -m pip install panel plotly pandas numpy" (without the quotation marks) 
Step 5: Run "py -m panel serve app.py --show" (without the quotation marks)
